## Contexto & Objetivo

**Site Atama Filmes:** Posicionar internacionalmente a produtora, mostrar portfolio (filmes, séries, streamings onde estão veiculados), prêmios e credibilidade global.

**Atama Lab:** Plataforma de cursos presenciais — regional (RS/Sul) com expansão nacional. Escalável para múltiplos formatos.

**Identidade de partida (logo):**
- Símbolo: mandala/labirinto circular em laranja queimado (~#D4612A)
- Wordmark: "atama" lowercase, bold, dark quase-preto (~#2D2D2D)
- Tagline: "FILMES" uppercase, letra-espaçamento amplo, em laranja/marrom
- Tom: autoral, artesanal, orgânico — não corporativo

## Padrões Identificados

### Visual Design

#### Paleta Cromática
| Cluster | Sites | Tom |
|---------|-------|-----|
| **Preto/escuro dominante** | O2 Filmes, Saigon, Landia, A24, Beham | Premium, cinematográfico, autoral |
| **Navy/azul escuro** | Cimarron, Beham | Corporativo-moderno |
| **Branco + tipografia forte** | Hyper Island, O2 Filmes (fundo), Yutopia | Educacional, clean, legível |
| **Cinza claro + tipografia** | Pure Cinema | Neutro, destaque pela forma |
| **Imagem/vídeo fullscreen** | Vulcana, Landia, A24 | Imersivo, cinematográfico |

**Padrão dominante:** O setor audiovisual evita cores vibrantes. Preto, branco, cinza e azul navy são a norma. Cor accent é exceção (A24 usa roxo/lilac como detalhe sutil; Pure Cinema usa gradiente como assinatura; Atama já tem o laranja no logo).

#### Tipografia
| Estilo | Sites | Mensagem |
|--------|-------|----------|
| **Display extrabold uppercase** | Saigon, Hyper Island, O2 | Impacto, autoridade |
| **Serif editorial** | A24, Yutopia | Premium, artístico, literário |
| **Sans-serif geométrica** | Landia, Cimarron, Vulcana | Moderno, clean |
| **Mix (serif logo + sans body)** | A24 | Distinção |

#### Uso do Espaço
- **Todos** os sites de maior qualidade têm espaço em branco generoso
- Nenhum site premium usa mais de 50% de área preenchida no hero
- O2 e Yutopia chegam a 90% de área vazia — espaço como luxo

### Navegação & UX

#### Menus
| Tipo | Sites | Quando funciona |
|------|-------|-----------------|
| **Hamburger simples** | Landia, A24, Pure Cinema | Sites com 4 ou menos seções |
| **Menu horizontal fixo** | Vulcana, Hyper Island, Cimarron | Sites com 5-7 seções |
| **Menu tipográfico central** | Saigon | Identidade forte como nav |
| **Menu no rodapé** | O2 Filmes | Portfolio imersivo |

**Padrão:** Produtoras com menos páginas usam hamburger. Escolas/plataformas com mais conteúdo usam menu horizontal.

#### Credibilidade Visual
Ordem de frequência nos melhores sites:
1. **Badges de festivais/streamings** (Berlinale, Netflix, Prime) — mais usados
2. **Nomes de diretores em destaque** (O2, Landia, Saigon) — autoria como credencial
3. **Associações institucionais** (Cimarron com Mediapro) — weight emprestado
4. **Imagens de pessoas reais** (Hyper Island) — humanização da experiência
5. **Nome sozinho carrega o peso** (A24) — só possível com brand consolidado

#### Internacionalização
Todos os sites com posicionamento global têm:
- Seletor de idioma visível (EN obrigatório)
- Copy bilíngue desde o hero
- Saigon: PT/EN/ES (Latam)
- Vulcana: PT/EN

## Recomendações para Atama

### A. Visual Identity

#### 1. Cor
**Decisão principal:** O logo laranja (#D4612A) é o maior diferencial de Atama entre os concorrentes.

> **Nenhum dos 11 sites usa laranja** como cor primária. Isso é uma oportunidade de diferenciação real.

- **Manter:** Laranja como accent único e reconhecível
- **Fundo:** Branco puro (#FFFFFF) ou off-white quente (~#FAF9F7) — complementa o laranja
- **Texto:** Quase-preto (~#1A1A1A) — não preto puro (menos agressivo, mais editorial)
- **Neutros:** Cinza claro (#F0F0F0), cinza médio (#999999)
- **Accent secundário:** Nunca concorre com o laranja — usar apenas quase-preto ou cinza

**Paleta Atama Recomendada:**
```
Brand Orange:  #D4612A  ← mantém identidade do logo
Near-Black:    #1A1A1A  ← texto principal
White:         #FFFFFF  ← fundo primário
Warm Off-White:#FAF9F7  ← fundo secundário (seções alternadas)
Gray-300:      #E0E0E0  ← bordas, divisores
Gray-600:      #757575  ← texto secundário
```

#### 2. Tipografia
**Descoberta:** Duas escolas dominam — Display extrabold (impacto, Saigon/Hyper Island) e Serif editorial (premium, A24/Yutopia).

**Recomendação para Atama:** Sistema híbrido, como A24

| Uso | Fonte | Justificativa |
|-----|-------|---------------|
| **Logo** | Mantém existente | |
| **Headlines** | **Playfair Display Bold** (serif) | Editorial, artístico — alinha com cinema autoral |
| **Subheadlines** | **Inter Semibold** (sans) | Legível, moderno, universal |
| **Body** | **Inter Regular** | Conforto de leitura |
| **Labels/Caps** | **Inter Medium, letter-spacing +2px** | Padrão do setor (Vulcana, Saigon) |

*Alternativa mais simples para V1: Inter em todos os pesos (mais fácil de implementar)*

#### 3. Imagery
**Descoberta:** Todos os melhores sites usam material próprio ou vídeo. Nenhum usa stock.

**Para site Atama Filmes:**
- Hero: Still cinematográfico de maior projeto (Beyond Us ou similar) fullscreen
- Portfolio: Grid com posters/stills dos filmes reais
- Team: Fotos reais de Rogério e Rose — em set ou ambiente autoral

**Para Atama Lab:**
- Hero: Foto de aula/oficina real (como Hyper Island) — pessoas interagindo
- Diferencial: Foto no espaço físico (Casa da Chácara) assim que disponível
- Sem stock de "pessoas estudando com laptop"

### B. Arquitetura & Navegação

#### Estrutura de Navegação Recomendada

**Para site Atama Filmes** (público: parceiros, festivais, distribuidores):
```
Filmes | Sobre | Lab | Contato     [EN] [PT]
```
- 4 itens + seletor de idioma
- "Lab" como item de menu = cross-sell claro para parceiros

**Para Atama Lab** (público: alunos, interessados em cursos):
- Entrada via `/lab` direto (TikTok, Insta bio)
- Navegação interna: Início | Cursos | Sobre o Lab | Inscrição
- CTA "Inscrever-se" sempre visível (sticky em mobile)

#### Internacionalização
Com objetivo de posicionamento internacional do site, **EN é obrigatório desde o lançamento V1**.
- Seletor PT | EN no canto superior direito (padrão do setor — ver Vulcana, Saigon, Cimarron)
- Hero copy em PT com tradução disponível no seletor
- Filmes: títulos bilíngues

## Síntese: O Posicionamento Visual Atama

Com base nos 11 sites analisados, Atama tem espaço para ocupar um posicionamento visual único:

> **"Cinema autoral brasileiro com presença global — branco, laranja e muita história."**

**Referências primárias a seguir:**
- **A24** → postura editorial, tipografia, detalhe refinado
- **Vulcana** → contexto RS, cinema autoral, internacionalização
- **Hyper Island** → modelo Lab: escola real com impacto humano
- **Yutopia** → minimalismo artístico, fundo claro, projetos como obras

**Referências do que evitar:**
- b_arco → visual genérico, baixa hierarquia, WhatsApp botão laranja flutuante
- Cimarron → corporativo demais para o tom autoral da Atama
