## Decisões de Arquitetura

### Produto
- **Um único domínio** para site Atama Filmes + Atama Lab (integrados)
- **Fundo branco** como identidade visual central
- **Português apenas** na plataforma Lab; **PT + EN** no site institucional
- **Mercado Pago** como gateway de pagamento preferencial (nacional, BRL)
- **Deadline V1: 31 de maio de 2026**

### Versionamento
| Versão | Escopo | Prazo |
|--------|--------|-------|
| V1 | Site simples + Lab funcional (sem motion design) | 31/05/2026 |
| V2 | Site completo com motion design (parallax, transições) + Lab melhorado | Jun/Jul 2026 |
| V3 | Lab evoluído (rodadas de negócio, mentorias) + site otimizado | Ago+ 2026 |

### Plataforma de cursos
- Decisão pendente entre: Hotmart, Kiwify ou solução própria
- Impacta: integração de pagamento, UX de compra, fluxo pós-matrícula
- **Bloqueio crítico**: decide o backlog técnico inteiro do V1

### Tech Stack
- A ser definido em [Setup] Setup técnico plataforma + CMS
- Restrição: ferramentas gratuitas ou baixo custo
