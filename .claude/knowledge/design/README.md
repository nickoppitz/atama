# Design — Identidade Visual e UX

Documentação de design, referências visuais, benchmark e padrões de interface.

## Arquivos

- **`benchmark-visual-ux.md`** — Análise visual e UX de sites/produtoras de referência
- **`benchmark-workspace/`** — Workspace com evals de UX, visual credibilidade, GUI padrões
- **`guidelines.md`** — Guia de identidade visual (fundo branco, cores, tipografia)

## Princípios

- **Fundo branco** é identidade central
- **Motion design é V2+** — nada de parallax ou transições no V1
- **Conversão site → Lab em <2 cliques**
