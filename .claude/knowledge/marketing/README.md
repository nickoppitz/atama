## Princípios do MA para este projeto

1. **Canal primário de aquisição é social outbound** (TikTok/Instagram), não busca orgânica — o site tem quase zero social traffic hoje
2. **Primeiras inscrições virão de ativação direta** — WhatsApp, rede de contatos do Rogério e Rose, não do site
3. **301 redirects são obrigatórios** na migração WordPress → Next.js — o domínio de 14 anos tem autoridade real que não pode ser desperdiçada
4. **Mobile-first é inegociável** — público Lab vem de redes sociais, usa celular
5. **Plausible + Search Console antes do go-live** — sem dados do dia 1, não há como otimizar pós-lançamento
6. **SEO orgânico é V2+** — não move inscrições para junho, mas não pode ser sabotado pela migração
