# A11y — Acessibilidade e Inclusão

Padrões, guias e validações para garantir que Atama seja acessível a todos.

## Arquivos

- **`regras-wcag.md`** — WCAG 2.1 AA compliance checklist
- **`color-contrast-guide.md`** — Requisitos de contraste de cores
- **`screen-reader-mobile.md`** — Testes com screen readers em mobile
- **`touch-target-guide.md`** — Tamanhos mínimos de touch targets

## Checklist de A11y para V1

- ✅ Contraste de cores WCAG AA em todas as seções
- ✅ Navegação por teclado funcional
- ✅ Labels em todos os inputs
- ✅ Alt text em imagens
- ✅ Touch targets ≥ 44x44px

Revise esses documentos antes de implementar qualquer feature no V1.
